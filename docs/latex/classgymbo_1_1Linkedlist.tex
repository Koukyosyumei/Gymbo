class gymbo\+::\+Linkedlist$<$ T $>$}

Doubly Linked List Implementation. 

The {\ttfamily \mbox{\hyperlink{classgymbo_1_1Linkedlist}{Linkedlist}}} class provides a simple implementation of a doubly linked list. It supports basic operations such as pushing elements onto the back, getting the length, accessing the element at the back, and popping the element at the back.
